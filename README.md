# 智慧教育网站

一个现代化的教育培训平台网站，采用响应式设计和最新的前端技术。

## 功能特性

### 🎯 核心功能
- **响应式设计** - 完美适配桌面、平板和移动设备
- **平滑滚动** - 优雅的页面导航体验
- **交互动画** - 丰富的CSS动画和过渡效果
- **移动端优化** - 专为移动设备优化的菜单和交互

### 🎨 设计特点
- **现代UI设计** - 采用渐变色彩和卡片式布局
- **优雅的动画** - 页面元素加载时的淡入动画
- **专业的配色** - 蓝紫色渐变主题，体现教育科技感
- **无障碍设计** - 支持键盘导航和屏幕阅读器

### ⚡ 技术特性
- **模块化结构** - HTML、CSS、JavaScript分离
- **性能优化** - 图片懒加载、代码分割
- **SEO友好** - 语义化HTML结构和元数据
- **渐进式增强** - 基础功能在不支持JS时也能使用

## 快速开始

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 或使用本地服务器运行：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```

## 自定义配置

### 修改主题颜色
在 `style.css` 中修改CSS变量：
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ff6b6b;
}
```

### 添加新课程
在 `index.html` 的课程网格中添加新的课程卡片：
```html
<div class="course-card">
    <h3>新课程名称</h3>
    <p>课程描述</p>
    <div class="course-meta">
        <span>时长：XX小时</span>
        <span>难度：初级</span>
    </div>
</div>
```
